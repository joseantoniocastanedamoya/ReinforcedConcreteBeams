About WinBeams

WinBeams is a structural software that is intended to design reinforced concrete beams using the ACI (American Concrete Institute) building code.

This application was created to serve as a helpful tool for professionals in the field of Civil Engineering and Architecture, as well as students of these two professions with the need to design any concrete beam quickly and efficiently.

WinBeams allows the user to enter the beam's span lengths, support types and different types of loads ranging from the beam's own weight, concentrated loads, applied couples, uniform loads and uniformly varying loads. The program allows the user to capture any number of load conditions and load combinations as well as the dimensions of the concrete section. The program also allows you to enter the compressive strength of concrete (f 'c) and the tensile strength of steel (fy). Once you have entered the minimum variables necessary to design a concrete beam the program is able to design the beam reinforcement and generate calculation reports.
